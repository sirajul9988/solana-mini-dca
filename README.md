# Solana Mini DCA

A streamlined, professional-grade script to automate Dollar Cost Averaging on the Solana blockchain. This tool interacts with the Jupiter Aggregator to ensure you always get the best price when swapping tokens.

## Features
* **Automated Swaps:** Set your interval and let the script handle the trades.
* **Jupiter Integration:** Uses the industry-standard Jupiter V6 API for optimal routing.
* **Simple Setup:** No complex folder structures; everything runs from the root.
* **Developer Friendly:** Clean code with inline documentation.

## Prerequisites
* Node.js (v16 or higher)
* A Solana wallet private key
* Some SOL for gas and swapping

## Quick Start
1. Install dependencies: `npm install`
2. Update the `config.js` with your private key and desired swap amount.
3. Run the script: `node index.js`
